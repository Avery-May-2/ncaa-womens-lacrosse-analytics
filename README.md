# NCAA Women's D1 Lacrosse Offensive Efficiency
This project scrapes team-level statistics from NCAA.com and complies them into a clean master dataset for analysis. The data is used to analyze offensive effiency through a variety of stat categories.

## Stats Collected
Assists
Goals
Draw Controls
Caused Turnovers
Free Postion %
Shots on Goal
Shots per Game

## Tools Used
Python
BeautifulSoup
Pandas
Google Colab
